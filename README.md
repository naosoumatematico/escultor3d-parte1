# projeto-sculptor

projeto da segunda e terceira unidade de programação avançada

# descrição

implementação da classe Sculptor, um escultor de modelos 3D com voxels que exporta a figura em formato OFF. pode ser visualizado em 3dviewer.net, em (esqueci qual eh o software do windows) e geomview.

# como compilar (linux)
no terminal:

g++ sculptor.h sculptor.cpp sculptor-main.cpp -o sculptor-main
